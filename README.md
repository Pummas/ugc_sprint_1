### Состав проекта

/ugc - cервис UGC на FastAPI  
/etl - ETL сервис

Команды
- `make format` отформатировать код
- `make lint` запустить линтер (flake8)
- `make run` запустить сервисы
- `make stop` остановить сервисы
- `make etl-logs` посмотреть логи etl
